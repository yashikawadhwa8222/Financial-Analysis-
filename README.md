# Financial Analysis Dashboard

## Project Overview

This dashboard provides insights into financial transactions, revenue trends, customer segments, transaction types, and regional performance. It helps stakeholders monitor key financial metrics and identify business trends for better decision-making.

## Tools Used

* Power BI
* Excel (Data Source)
* DAX Measures
* Data Cleaning & Transformation

## Key Metrics

* Total Amount
* Total Transactions
* Average Transaction Value
* Total Fees
* Total Tax

## Dashboard Features

* Monthly Revenue Analysis
* Transaction Status Distribution
* Customer Segment Analysis
* State-wise Revenue Analysis
* Transaction Type Performance
* Gender-wise Revenue Distribution
* Dynamic Filters (Year, Occupation, Category)

## DAX Measures Used

* Total Amount
* Total Transactions
* Average Transaction Value
* Total Fees
* Total Tax
* Year-over-Year Growth %

## Business Insights

* Retail customers contribute the highest revenue.
* Maharashtra generates the highest transaction amount.
* Successful transactions account for the majority of total revenue.
* Revenue is distributed almost equally between male and female customers.

## Dataset

This project uses a sample banking dataset containing transaction details, customer demographics, transaction categories, fees, taxes, and regional information.

## Purpose

The objective of this dashboard is to demonstrate data analysis, visualization, and business intelligence skills using Power BI.
